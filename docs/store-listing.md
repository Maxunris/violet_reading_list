# Store Listing Copy

## Name

Violet Reading List

## Short Description

Save links with folders, tags, favorites, pins, and domain search in a purple-first reading list for Chrome and Firefox.

## Full Description

Violet Reading List is a cleaner way to save pages for later when a plain backlog is no longer enough.

Instead of throwing every link into one long list, Violet helps you organize saved pages with folders on the left, tag-based filtering, favorites, pinned items, and fast search across titles, descriptions, URLs, and domains.

### What you can do

- Save the current tab in one click
- Save links from the page context menu
- Organize entries into readable folders
- Create, rename, delete, and filter by tags
- Mark important items as favorites
- Pin items so they stay at the top
- Search by text or narrow the list by domain
- Edit titles and descriptions after saving
- Export your library to JSON and import it later

### Why Violet feels different

The popup is designed like a compact reading workspace rather than a flat dump of bookmarks. Folders stay visible on the left, filters stay close to the list, and each entry shows human-friendly metadata, including title, description, domain, and favicon when available.

### Browser support

Violet Reading List is built as a cross-browser WebExtension and is designed for Chromium-based browsers and Firefox.

Turn a messy save-for-later pile into a structured reading library with folders, tags, favorites, and fast domain-aware search.
